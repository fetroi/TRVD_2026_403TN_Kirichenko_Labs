# Лабораторна робота №5

**Тема:** Веб-додаток для управління футбольним клубом (Frontend UI/UX)
**Студент:** Кириченко Сергій Васильович
**Група:** 403-ТН

---

### Мета роботи
Реалізувати клієнтську частину системи управління футбольним клубом, забезпечити зручний інтерфейс (UI) для керування складом команди, інтегрувати систему фільтрації даних та впровадити візуальні ефекти для покращення досвіду користувача (UX).

---

### 1. Архітектура фронтенду
Проєкт реалізований як сучасний SPA на базі React.js (Vite).
* **Single-file Architecture:** Для забезпечення високої швидкості розробки та уникнення конфліктів стилів, основна логіка, стан та стилізовані компоненти консолідовані у головному модулі App.jsx.
* **State Management:** Управління станом реалізовано через React-хуки `useState` (для збереження списку гравців, фільтрів та станів редагування) та `useEffect` (для синхронізації з базою даних Supabase).

---

### 2. Інтерфейс автентифікації та Dashboard
* **Auth UI:** Реалізовано форму входу (компонент Auth.jsx), яка інтегрована з Supabase Auth.
* **Smart Dashboard:** Після входу тренер бачить головну панель, що включає:
    * **Header:** З назвою клубу та кнопкою швидкого виходу.
    * **Stats Cards:** Блок інтерактивної статистики, що автоматично підраховує кількість гравців «Всього», «В строю» та «Травмовані» на основі даних із БД.

---

### 3. Функціонал управління (UI/UX)
* **Динамічна фільтрація:** Впроваджено систему швидких фільтрів за амплуа (Воротарі, Захисники, Півзахисники, Нападники).
* **Inline Editing:** Можливість редагування номера або позиції гравця безпосередньо в списку без видалення запису.
* **Toggle Status:** Кнопки швидкої зміни стану здоров’я ( 🚑 / ✅ ) з кольоровою індикацією статусів.

---

### 4. Візуальні ефекти та Анімація
Для покращення UX було впроваджено:
* **CSS Animations:** Використання `@keyframes fadeInUp` для плавної появи рядків таблиці.
* **Cascade Effect:** Кожен рядок таблиці з'являється з невеликою затримкою (`animation-delay`), що створює ефект професійного динамічного завантаження.
* **Modern Styling:** Використання темної теми (Dark Mode), ефектів напівпрозорості (`backdrop-filter: blur`) та округлених кутів.

---

### 5. CRUD Операції в інтерфейсі
* **Додавання:** Форма з випадаючим списком позицій для запобігання помилкам введення.
* **Виведення:** Таблиця з автоматичним сортуванням за номером гравця.
* **Видалення:** Оптимізована функція видалення запису в один клік для прискорення роботи.

---

### Як запустити
1. Встановити залежності: `npm install`
2. Запустити проєкт: `npm run dev`
3. Відкрити локальну адресу (зазвичай `http://localhost:5173`)
